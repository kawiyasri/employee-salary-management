# Bug Report 001: Data Gaji Page Shows Blank Screen

## Bug ID
BUG-001

## Title
Data Gaji page becomes blank after navigation.

## Severity
High

## Priority
High

## Environment
- Application: SiPeKa HRMS
- Frontend: http://localhost:5173
- Backend: http://localhost:5000
- Browser: Google Chrome Version 149
- User Role: Admin (Aldi)

## Preconditions
User is logged in as an administrator.

## Steps to Reproduce

1. Login using admin credentials.
2. Navigate to "Transaksi".
3. Click on "Data Gaji".
4. Observe the page.
5. Refresh the browser.

## Expected Result

The salary management page should display salary records and salary processing options.

## Actual Result

The page becomes blank and refreshing does not recover the screen.

## Impact

Payroll administrators cannot access salary information or process salaries, potentially delaying employee payments.

Screenshot attached in screenshots folder.

Open
